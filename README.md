# 🌤️ Weather App

A responsive weather application that shows the current conditions and a **3-day forecast** for any city, built with vanilla JavaScript and the WeatherAPI service.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

## 📖 Overview

Search for any city and instantly get live weather data — temperature, weather status with icons, wind, humidity and a 3-day forecast. Data is fetched asynchronously from the **[WeatherAPI](https://www.weatherapi.com/)** REST service using the Fetch API and `async/await`.

## ✨ Features

- 🔎 Search weather by city name
- 📅 Current day + 3-day forecast
- 🌡️ Temperature, weather condition, wind speed and humidity
- 🖼️ Dynamic weather icons for each condition
- 📱 Fully responsive layout

## 🛠️ Tech Stack

- HTML5 & CSS3
- Vanilla JavaScript (ES6+, Fetch API, async/await)
- [WeatherAPI](https://www.weatherapi.com/) REST API

## 🚀 Getting Started

```bash
git clone https://github.com/gehad-ahmed/weather.git
cd weather
```

Then open `index.html` in your browser (or use the VS Code **Live Server** extension).

> 🔑 **Note:** The app calls WeatherAPI using an API key. For your own deployment, get a free key from [weatherapi.com](https://www.weatherapi.com/) and replace it in `js/index.js`.
